# AVO FnC 생산 관리 시스템

## 📌 프로젝트 개요
**제조 공정 시각화 웹 인터페이스**  
5개 작업 라인에 대한 실시간 생산 관리 시스템으로, 오일/마감재/라벨 선택과 자재 정보 표시 기능을 포함

## ✨ 핵심 기능
| 기능 영역 | 세부 내용 | 지원 옵션 |
|-----------|-----------|-----------|
| **업체 선택** | 업체 선택 | |
| **상단 루땡** | 7종 컬러 마감 처리 | 금색/붉은색/검정색 |
| **오일 선택** | 업체별 오일 선택, 선택하면 라벨과 오일 이미지 나옴 |  |


## 🛠 기술 스택
- **HTML5**  
  > 테이블 기반 다중 작업 라인 레이아웃

- **CSS3**  

- **JavaScript(ES6)**  

- 동적 옵션 생성(Array.map())

- 이벤트 리스너 기반 실시간 업데이트
